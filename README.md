# Arcus
A simple game made in Python using pygame

### Requirements :-
1. Python(2 or 3)
2. pygame 
      install it by typing

      `sudo apt-get install python-pygame`

       OR

      `pip install pygame`
  
## How to play :-
1. Clone the repo
2. Run `python main.py`

### Controls :-

1. Give direction to arrow using mouse
2. Drag the arrow to increase velocity
